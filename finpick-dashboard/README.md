# FinPick AI Voice Sales Dashboard

A modern React dashboard for managing your FinPick AI Voice Sales System.

## Features

- **Real-time Dashboard**: View calls, conversions, and hot leads at a glance
- **Lead Management**: Track lead status (HOT/WARM/COLD) with detailed information
- **Activity Feed**: Monitor AI calls, WhatsApp messages, and CRM updates
- **Performance Charts**: Weekly call volume and conversion tracking
- **System Configuration**: View integration status for CRM, WhatsApp, and voice agent

## Tech Stack

- React 18
- Vite
- Lucide React Icons
- Modern CSS with gradients and animations

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
npm run preview
```

## Access the Dashboard

The dashboard is now hosted and accessible at:
- **Local**: http://localhost:8080
- **Network**: http://21.0.3.107:8080

## Integration Points

This dashboard is designed to connect with:
1. **Retell AI** - For AI voice call management
2. **ElevenLabs** - For Hindi/Marathi voice synthesis
3. **Zoho CRM** - For lead management
4. **WhatsApp Business API** - For automated follow-ups

## Customization

Edit `src/App.jsx` to:
- Connect real API endpoints
- Update configuration values
- Add new features like lead filtering, campaign management
